# ⚡ Быстрый старт Neira v0.6.1

## 🚀 Первый запуск (3 команды)

```bash
# 1. Настройка (один раз)
./setup.sh

# 2. Запуск
./start.sh

# 3. Остановка (когда закончите)
./stop.sh
```

Вот и всё! 🎉

---

## 📋 Что делают скрипты

### `./setup.sh` — Первоначальная настройка
Выполняется **один раз** при первой установке:
- ✅ Проверяет Python 3.10+
- ✅ Устанавливает зависимости (requests)
- ✅ Устанавливает Ollama (если нужно)
- ✅ Скачивает модели (~10 GB)
- ✅ Создаёт структуру проекта

### `./start.sh` — Запуск Neira
Используется для **каждого запуска**:
- ✅ Проверяет все зависимости
- ✅ Автоматически запускает Ollama (если не запущен)
- ✅ Проверяет модели
- ✅ Запускает Neira

### `./stop.sh` — Остановка
Останавливает Ollama сервер (опционально):
- ✅ Корректно завершает Ollama
- ✅ Освобождает ресурсы

## 💡 Альтернативный запуск (вручную)

Если скрипты не работают, запускайте вручную:

```bash
# Терминал 1: Запустить Ollama
ollama serve

# Терминал 2: Запустить Neira
python3 main.py
```

## 🐛 Проблемы?

### Ollama не запускается
```bash
# Проверьте процесс
ps aux | grep ollama

# Запустите вручную
ollama serve
```

### Модели не найдены
```bash
# Скачайте модели
ollama pull qwen2.5-coder:7b
ollama pull ministral-3:3b
```

### Python ошибки
```bash
# Проверьте версию
python3 --version  # Должна быть 3.10+

# Установите зависимости
pip install requests
```
