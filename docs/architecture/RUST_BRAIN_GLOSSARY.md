# Rust Brain Glossary — ключевые термины

> Цель документа: зафиксировать устойчивые определения терминов, чтобы обсуждения архитектуры Rust‑ядра были однозначными.

## Термины

- **Pathway (путь обработки)** — направленный маршрут передачи сигналов между компонентами мозга (например, от распознавания намерения к синтезу ответа). Англ. пояснение: *pathway = directed processing route between subsystems*.
- **Tier (уровень)** — фиксированный слой архитектуры с чёткими границами ответственности и контрактами данных. Англ. пояснение: *tier = layered boundary with explicit contracts*.
- **Wisdom Core (ядро мудрости)** — устойчивый слой знаний и принципов, который не зависит от контекста конкретного диалога и служит ориентиром для решений.
- **Response Synthesizer (синтезатор ответа)** — модуль, который формирует итоговый ответ на основе сигналов от пути обработки, контекста и политик качества.
- **Learning Loop (контур обучения)** — циклический механизм улучшения, который собирает обратную связь, обновляет правила и переносит новые выводы в долгосрочные представления.
- **LLM Consultant (LLM‑консультант)** — внешний или вспомогательный модуль, который предоставляет вероятностные рекомендации, но не является источником истинных правил.
- **Organ System (система органов)** — совокупность специализированных модулей, отвечающих за отдельные функции (например, коммуникации, память, безопасность).
- **Intent Recognizer (распознаватель намерения)** — компонент, который выделяет цель пользователя и контекст задачи, формируя структурированное намерение для следующих слоёв.
