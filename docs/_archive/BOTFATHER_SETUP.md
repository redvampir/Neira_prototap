# Установка команд для Telegram бота через @BotFather

## Шаг 1: Откройте @BotFather

1. Найдите @BotFather в Telegram
2. Отправьте `/mybots`
3. Выберите вашего бота
4. Нажмите `Edit Bot` → `Edit Commands`

## Шаг 2: Вставьте команды

Скопируйте и вставьте следующий текст в @BotFather:

```
start - Приветствие и быстрый старт
help - Справка по командам
myname - Установить своё имя
stats - Статистика бота
memory - Просмотр памяти
context - История диалога
clear_context - Очистить контекст
rhythm - Стабилизатор настроения
imagine - Генерация изображений
vision - Статус распознавания
```

## Шаг 3: Сохраните

Нажмите "Send" в @BotFather - готово! ✅

---

## Дополнительно: Команды для администратора

Если хотите установить расширенный список команд для себя как администратора:

1. В @BotFather отправьте `/setcommands`
2. Выберите вашего бота
3. Выберите scope: `chat_administrators` 
4. Вставьте полный список:

```
start - Приветствие и быстрый старт
help - Справка по командам
myname - Установить своё имя
stats - Статистика бота
memory - Просмотр и управление памятью
experience - Журнал опыта
context - История диалога
clear_context - Очистить контекст
clear - Полная очистка памяти
rhythm - Стабилизатор настроения
cortex - Статистика Cortex v2.0
auth - Авторизация пользователя
admin - Панель администратора
learn - Обучение на примерах
learn_auto - Автономное обучение
self - Самоанализ системы
organs - Статус органов
grow - Создание новых органов
code - Управление кодом
imagine - Генерация изображений
vision - Статус распознавания
```

---

## Проверка

После установки команд:
1. Откройте чат с ботом
2. Нажмите `/` — должно появиться меню с командами
3. Обычные пользователи видят 10 команд
4. Админ видит 21 команду

✅ Готово!
